Zig-Zag-Zoom ⚡

Fast-paced arcade madness where the only rule is: keep moving...

Unity C# Arcade Procedural Generation

A fast-paced arcade game built in Unity (C#) — I designed the input-responsive vehicle controls,

the procedurally generated track system, and a collision detection setup that makes one wrong zig genuinely catastrophic.

Simple rules. Zero mercy.

Under the Hood 🔧

  • Engine & Language: Unity · C#
  • Vehicle Controls: Built input-responsive movement system with smooth directional transitions and speed scaling.
  • Procedural Generation: Designed a dynamic track generation system that creates new obstacle layouts in real time
  • Collision Detection: Implemented precise collision logic tied to game state and scoring system
  • Difficulty Scaling: Speed and obstacle frequency escalate progressively — no mercy mode kicks in faster than you'd like
  • Coin & Scoring System: Pickup logic, score tracking, and high score persistence
