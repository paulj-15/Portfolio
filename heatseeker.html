Heatseeker Multiplayer ⚽

Think Pong, but angrier and multiplayer...

A real-time multiplayer arcade game built in Unity (C#) — I implemented the netcode, designed the deflection mechanics,

and built a scoring system that makes sure bragging rights are mathematically earned, not given.

A ball that never calms the hell down included at no extra charge.

Under the Hood 🔧

  • Engine & Language: Unity · C#
  • Multiplayer Architecture: Implemented real-time multiplayer netcode for two-player simulatneous gameplay
  • Deflection Mechanics: Designed and scripted the ball physics and deflection logic — speed increases on each hit, angles calculated on impact
  • Scoring System: Built score tracking, win condition logic, and session management
  • Game State Management: Scripted lobby, match, and end-screen state transitions with smooth player flow
  • Input Handling: Implemented responsive player input for both local and networked sessions
