Scalar 🏃

An endless runner where obstacles fly at you faster than your morning deadlines...

An endless runner built in Unreal Engine 5 — I implemented the procedural obstacle generation system, the coin pickup logic,

and a difficulty scaling curve that escalates faster than your morning deadlines.

How far can you get before the game wins? Spoiler: the game always wins.

Under the Hood 🔧

  • Engine & Language: Unreal Engine 5 · Blueprints
  • Procedural Generation: Built a dynamic obstacle spawning system that generates new layouts in real time using Blueprints
  • Difficulty Scaling: Designed a progressive speed and frequency curve — the game gets harder faster than you're ready for
  • Player Movement: Scripted responsive character movement with frame-accurate input handling
  • Coin & Pickup System: Implemented collectible logic, score tracking, and high score persistence
  • Game State Management: Built start, play, and game-over state transitions using Blueprint event graphs
